# よくある質問
## 結局どのコマンドを使えばいいんですか！？
<!--Todo: コマンドリファレンスへのリンクを貼る-->
`新しいパネルを作りたい！パネルを複数作りたい！` => createコマンド

`パネルに役職を追加したい！（選択肢を増やしたい）` => addコマンド

`パネルの役職を削除したい！（選択肢を減らしたい）` => remove コマンド

`パネルのタイトルや色を変更したい！` => editコマンド

`パネルそのものを消したい` => deleteコマンド または、パネルのメッセージ自体を削除しても構いません。

## 「役職パネルv3」に「役職パネルv2」のデータを移行できますか？
はい、移行できます。

「v2パネル移行コマンド」をお使いください。

## 「役職の付与に失敗しました。BOTの一番上よりも高い役職をつけようとしてるかも？」と表示された
「サーバー設定」→「ロール」

その後、「役職パネルv3」という役職が、つけたい役職よりも上になっているか確認してください

## なぜかパネルの役職がdeleted-roleになるんだけど！
Discord側のバグです。

もし正常に役職を付けられているのであれば役職パネルの問題ではありませんので、Discord運営に問い合わせてみてください。
